# Sail-Track

This is an in development project

## Install
Run command in terminal

``` sh
$ npm install
```

or

``` sh
$ yarn install
```

## Dev

``` sh
$ npm run dev
```

or

```
$ yarn dev
```

Then open http://localhost:8000/ in browser, or http://xxx.xxx.xxx.xxx:8000 on mobile devices with the host's local network IP address.
